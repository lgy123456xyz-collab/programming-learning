# 🃏 港式梭哈 — Hong Kong Stud Poker

一个完整的港式梭哈扑克牌游戏，带有自我训练 AI 对手。

## 📁 项目结构

```
hk_stud_poker/
├── main.py          # PyQt5 图形界面主程序
├── game_engine.py   # 游戏逻辑引擎（发牌、下注、牌型判断）

pip install PyQt5

```

## 🎮 游戏规则

### 港式梭哈基本规则
- **5张牌梭哈**：每位玩家获得5张牌
- **第1张**：暗牌（只有自己可见）
- **第2-5张**：依次发出明牌（所有人可见）
- **4轮下注**：每发一张明牌后进行一轮下注
- **先手规则**：每轮由持有最大明牌的玩家先行动
- **底注**：每局开始前所有玩家缴纳底注

### 行动选项
| 操作 | 说明 |
|------|------|
| 弃牌 | 放弃手牌，退出本局 |
| 过牌 | 不下注（仅无人加注时可用） |
| 跟注 | 跟上当前最高注 |
| 加注 | 提高注码（可拖动滑块设置金额） |
| 全押 | 押上所有筹码 |

### 牌型大小（从大到小）
```
皇家同花顺 > 同花顺 > 四条 > 葫芦 > 同花 > 顺子 > 三条 > 两对 > 一对 > 高牌
```

## 🤖 AI 系统

### 技术架构
AI 对手采用三层混合策略：

1. **Q-Learning（强化学习）**
   - 状态空间编码：牌力等级 × 街数 × 底池赔率 × 对手数 × 位置
   - 动作空间：弃牌/过牌/跟注/加注/全押
   - 经验回放缓冲区（2000条）
   - ε-贪心探索策略，随对局自动衰减

2. **Monte Carlo 模拟**
   - 对当前手牌胜率进行概率估算
   - 根据难度级别决定模拟频率（新手10% → 大师100%）
   - 200次随机模拟估算获胜概率

3. **对手建模（Opponent Modeling）**
   - 跟踪每个对手的行动历史
   - 计算: VPIP（主动入池率）、激进度、弃牌率
   - 当对手偏保守时，AI 会增加虚张声势（Bluff）频率

### AI 难度
| 难度 | Monte Carlo频率 | 探索率 | 说明 |
|------|----------------|--------|------|
| 新手 | 10% | 50% | 频繁随机行动 |
| 普通 | 30% | 25% | 基础策略 |
| 高手 | 70% | 10% | 强力策略 |
| 大师 | 100% | 5% | 近乎最优 |

### 自我训练
- 点击主菜单「AI 自我训练」
- 设置训练局数（100~5000局）
- AI 通过 3 位 AI 相互对弈进行强化学习
- 模型保存至 `trained_ai.json`
- 下次游戏时自动加载训练后的模型

## 💡 开发说明

### 奖励函数
- 每局最终奖励 = 净盈亏 / max(|净盈亏|, 1)，范围 [-2, +2]
- 每存活一条街获得 +0.01 小奖励（鼓励坚持）
- 弃牌时获得 0 奖励

### 状态编码示例
```
s7_st3_po1_op1_L_ag0
 │   │   │   │  │  └─ 是否为上一个加注者
 │   │   │   │  └──── 位置(E/M/L/D)
 │   │   │   └─────── 对手数(0-2)
 │   │   └─────────── 底池赔率(0-3)
 │   └─────────────── 街数(1-4)
 └─────────────────── 牌力桶(0-9)
```
