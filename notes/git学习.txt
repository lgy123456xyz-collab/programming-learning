配置用户名，邮箱
git config --global user.name "xxx"
git config --global user.email "xxx@xxx"

检查
git config --list

密钥生成
ssh-keygen -t rsa -b 4096 -C "xxx@xxx"

本地仓库配置
git init   //创建.git
git remote add origin 网址  //链接
git pull origin master  //拉取文件
git config core.ignorecase false //使能大小写敏感

添加.gitignore 文件

/**/*.exe
/**/cmake-build-debug
/**/.idea



git clone 网址 //克隆仓库

代码推送 
先拉取 git pull origin master
git add . 将所有修改加入暂存区
git commit -m "版本名字"  同步到本地仓库
git push origin master 上传仓库到云端 


修改本地主分支名称
git branch -m master main
